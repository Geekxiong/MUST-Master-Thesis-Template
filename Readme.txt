稍微整理了一下学姐给的模板，其实目录结构还是比较简单的。
本意是想让大家花跟多的时间在写论文上面，而不是调模板上面。
如果模板有什么问题的话可以联系我，请见谅。

注意事项：
1. 模板的实例章节只有两个，你可以自行在chapters文件夹中再新建.tex文件来新增章节，记得要在 graduation-thsis-2020.tex 文件中的倒数第六行那里用\include{}来引入格式和示例一样。
2. 一般会用到的引用，表格，公式，图片的例子也在模板里简单的各加了一个。
3. 引用的格式使用的是BibTeX，一般谷歌学术上面都有。 文件是 other_pages 文件夹下的  bibliography.bib。
4. 最后有一个个人简历需要填写的， 需要先在other_pages文件夹下找到 personal_resume.docx  编辑成你的信息，然后再转成PDF文件，文件名必须为personal_resume.pdf。
5. 图片一般放在Figs文件夹下面，新增图片的时候可以参考示例。
6. 编译器需要选择 xelatex 不然编译不了。
